Selector Ejemplo Descrición del ejemplo CSS
.class .intro Selecciona todos los elementos de la clase intro: class="intro" 1
#id #firstname Selecciona el elemento cuyo id sea firstname: id="firstname" 1
* * Selecciona todos los elementos 2
element p Selecciona todos los elementos <p> 1
element,element div, p Selecciona todos los elementos <div> y todos los elementos <p> 1
element element div p Selecciona todos los elementos <p> dentro de elementos <div> 1
element>element div>p Selecciona todos los elementos <p> cuyo padre sea un <div> 2
element+element div+p Selecciona todos los elementos <p> ubicados inmediatamente despúes de un <div> 2
[attribute] [target] Selecciona todos los elementos con atributo target 2
[attribute=value] [target=_blank] Selecciona todos los elementos con target="_blank" 2
[attribute~=value] [title~=flower] Selecciona todos los elementos que contengan la palabra "flower" en su atributo title 2
[attribute|=value] [lang|=en] Selecciona todos los elementos con un atributo lang que empiece por "en" 2
:link a:link Selecciona todos los enlaces no visitados 1
:visited a:visited Selecciona todos los enlaces visitados 1
:active a:active Selecciona los enlaces activos 1
:hover a:hover Selecciona los enlaces que tengan el ratón encima de ellos 1
:focus input:focus Selecciona los elementos input que tengan el foco 2
:first-letter p:first-letter Selecciona la primera letra de cada párrafo <p> 1
:first-line p:first-line Selecciona la primera línea de cada párrafo <p> 1
:first-child p:first-child Selecciona todos los elementos <p> que son el prime hijo de su elemento padre 2
:before p:before Inserta contenidos antes de los contenidos de cada párrafo <p> 2
:after p:after Inserta contenidos despues de los contenidos de cada párrafo <p> 2
:lang(language) p:lang(it) Selecciona todos los elementos <p> element with a lang attribute value starting with "it" 2
