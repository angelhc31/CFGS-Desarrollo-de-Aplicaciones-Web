Propiedades de fondos
  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position
  • color de fond
  • imagen de fondo
  • repetición de la imagen de fondo (repeat/no-repeat)
  • define el desplazamiento de la imagen de fondo (fixed/scroll)
  • posición de la imagen de fondo (left, right, top, bottom)
body {background-color:#b0c4de;}
body {background-image:url('gradient2.png');}
body {background-repeat:no-repeat;}
body {background-attachment:fixed;}
body {background-position:right top;}
Propiedades relacionadas con el formato de texto
  • color
  • text-align
  • text-decoration
  • text-transform
  • text-indent

Otros: letter-spacing, line-height, vertical-align, word-spacing
  • Establece el color
  • Establece la alineación (left, right, center, justify)
  • Decoración (overline, line-through, underline, blink)
  • Transformación (uppercase, lowercase, capitalize)
  • Indentación o sangría de primera línea

 

body {color:blue;}
h1 {text-align:center;}
h3 {text-decoration:underline;}
p {text-transform:lowercase;}
p {text-indent:50px;}
h1 {letter-spacing:2px}
p {word-spacing:30px;}


Propiedades relacionadas con el formato de fuente
  • font-family
  • font-style
  • font-size
  • font-weight
  • Define la fuente empleada en el selector
  • El estilo de fuente (normal, italic, oblique)
  • Tamaño de la fuente (px, em, %)
  • Define el peso o espesor del texto, como la negrita
p{font-family:"Times New Roman", Times, serif;}
p {font-style: normal;}
h1 {font-size: 40px;}
a {font-weight:bold;}
Propiedades para enlaces (pseudo-clases)
  • a:link
  • a:visited
  • a:hover
  • a:active

definen cualquier propiedad de las anteriores asociadas a los diferentes estados de un enlace: no visitado, visitado, ratón encima o enlace activo

 

a:link {text-decoration:none;}
a:visited {color:#00FF00;}
a:hover {background-color:#FFFF85;}
a:active {color:#0000FF;}
Propiedades relacionadas con los bordes y líneas
  • border
  • border-color
  • border-style
  • border-width
  • border-bottom (top/left/right)
  • border-bottom-color (style/width)

  • outline
  • outline-color
  • outline-style
  • outline-width
  • Establece propiedades para los bordes de un elemento (anchura, estilo y color)
  • Establece especificamente el color para todos los bordes
  • Establece especificamente el estilo (solid, dotted) para todos los bordes
  • Establece especificamente la anchura (en px) para todos los bordes
  • Establece propiedades para un borde (inferior, superior, izquierdo o derecho)
  • Establece alguna propiedad (color, style, width) para un borde (inf, sup, izq, der)

  • Define todas las propiedades para las líneas exteriores en una sola declaración
  • Define el color de las líneas exteriores
  • Define el estilo de las líneas exteriores
  • Define la anchura de las líneas exteriores

p { border:5px solid red; }
h1{
     border-style:solid;
     border-left-color:#ff0000}
p {
     border-style:double;
     border-right-width:15px;
     border-left-width:10px;
     }


p {outline:#00FF00 dotted thick;}
p {
     outline-style:dotted;
     outline-width:5px;
     }

Propiedades de listas
  • list-style-type
  • list-style-position
  • list-style-image
  • list-style
  • Define el tipo de marcador para los elementos de lista (square, circle, disc, lower-roman, upper-roman, lower-latin, upper-latin, decimal)
  • Define la posición del marcador del listado (inside, outside)
  • Permite estableder una imagen como elemento de lista
  • Define todas las propiedades de lista en una sola sentencia
  • list-style-type:upper-roman
  • list-style-position:inside;
  • list-style-image:url('sqpurple.gif');
  • list-style: square url("sqpurple.gif");
Propiedades de dimensión
  • height
  • width
  • max-height
  • max-width
  • min-height
  • min-width
  • Define la altura del elemento
  • Define la anchura del elemento
  • Define la altura máxima del elemento
  • Define la anchura máxima del elemento
  • Define la altura mínima del elemento
  • Define la anchura mínima del elemento

 

p.especial {
     height:100px;
     width:100px;
     }
div {
     max-width: 1000px;
     min-width: 300px;
     }

Propiedades de márgenes y relleno
  • margin
  • margin-top (right, bottom, left)

  • padding
  • padding-top (right, bottom, left)
  • Permite definir todos los margenes a la vez en sentido horario
  • Permite definir cada márgen de forma independiente

  • Permite definir todos los rellenos a la vez en sentido horario
  • Permite definir cada espacio de rellenos de forma independiente

Notas: se pueden usar 4, 3, 2 o una unidad definiéndose en sentido horario:

  • margin: 10px 5px 3px: estable top a 10px, y bottom a 5px y y right a 3px
  • padding: 10px 5px: estable top y bottom a 10px y right y left en 5px

p {
     margin:2cm 4cm 3cm 4cm;
     padding:10px;
     }


div {
     margin:10px;
     padding:5px 25px;
      }
Propiedades de posición
  • bottom, top, left , right
  • visibility
  • z-index
  • clip
  • cursor
  • display
  • float
  • clear
  • overflow
  • position

 

 
  • Especifica la posición inferior, superior, izquierda y derecha de un elemento
  • Define la visibilidad de un elemento (visible/hidden)
  • Define la profundidad de un elemento en eje Z
  • Proporciona un recorte en el contenido del elemento posicionado
  • Especifica el tipo de cursor a mostrar
  • Especifica cómo debe ser mostrado cierto elemento (inline, block, inherit, none)
  • Especifica si el elemento flota o no y en qué posición (left/right)
  • Especifica que lado o lados del elemento no permitirán flotantes (left, right, both)
  • Especifica cómo se comporta el contenido de un item (scroll, visible, hidden, auto)
  • Especifica el tipo de posición de un elemento:
    • static: por defecto, los items se posicionan por el orden en que aparecen
    • absolute: los items se posicionan en función del primero no estático
    • fixed: los items se posicionan respecto a la ventana del navegador
    • relative: los items se posicionan en relación a donde se encuentran
    • inherit: heredada del elemento dentro del que se encuentra

  
Propiedades de tablas
  • border-collapse
  • border-spacing
  • caption-side
  • empty-cells
  • table-layout

 
  • Colapsa los bordes (pasan de ser dobles a sencillos de una línea por defecto); valores collapse/separate
  • Establece la distancia entre bordes de una tabla
  • Permite usar un pié de tabla y su posición (top/bottom)
  • Muetra o esconde las celdas vacías (hide, show)
  • Establece si una tabla debe o no redimensionarse respecto a sus contenidos (auto, fixed)
  • border-collapse:collapse;
  • border-spacing:10px 50px;
  • caption-side:bottom;
  • empty-cells:hide;
  • table-layout:fixed;
